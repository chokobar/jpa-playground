# JPA 연습 프로젝트

## 📖 개요
처음으로 JPA를 공부하며 만든 연습용 프로젝트입니다.  
Spring Boot + JPA 기반으로 간단한 게시판/회원 시스템을 구현했습니다.

---

## 🛠 기술 스택
- Java 17
- Spring Boot 3.4
- Spring Data JPA
- MySQL Database
- Thymeleaf
- Gradle


## 🔍 주요 기능
- 회원가입 / 로그인
- 게시글 CRUD
- Entity 간 연관관계 설정 (OneToMany, ManyToOne 등) [진행중]


## 🧪 학습 포인트
- JPA의 기본 CRUD 동작 확인
- 연관관계 매핑 및 지연 로딩 테스트
- JPQL / Native Query 실험
